下面视频展示的这份PPT 是由Tu+、Namroc、Metcalf Anything 三位协作打造而成的，长达450页，其它相关技术参数如下：

3 animators 三位闲人

3 locations 三地协作

3 days (sort of) 三天时间

1 Google Doc 仅用谷歌文档完成

全部过程就只用到了Google Docs ，无任何视频动画软件，纯手工完成。不过PPT 的内容本身没有什么实际含义，主要由点线面及各种颜色构成，当成普通的定格动画播放即可。

该视频是为了参加Google Demo Slam 的活动，在YouTube 上的播放已超120万次—— Google Demo Slam: Epic Docs Animation （想看的话你懂的），目前就所有的参加作品来看，就这段遥遥给力。

通过各种方法都没有搞到原始的PPT 文档，只导出了一个PDF（下载/3.61MB），强烈建议下载下来观赏。
