Reaction Native for Windows和Mac

2020-05-13 05:45:23

使用基于Javascript和Reaction的一致开发者体验，Reaction Native使您能够在多个平台上构建世界级的本机应用程序体验。

Reaction Native for Windows&；Mac为Windows 10 SDK和MacOS 10.12 SDK带来了Reaction Native支持。有了这一点，您可以使用Javascript为Windows 10支持的所有设备(包括PC、平板电脑、2合1、Xbox、混合现实设备等)以及MacOS台式机和笔记本电脑生态系统构建原生Windows应用程序。

您可以根据需要以任何方式使用React Native for Windows&；Mac，包括以下内容：

使用Reaction Native for Windows&；Mac从头开始创建完整的Windows 10和MacOS 10.12应用程序。

将Reaction Native for Windows&Mac组件添加到现有的本机Windows 10或MacOS 10.12项目中。

我们正在积极开发Reaction Native for Windows，以配合REACT-Native内核的发展。有关这一努力的详细信息和未来几个月的投资路线图，请参阅Roadmap 2019博客。

REACT Native for Windows现在支持REACT Native版本0.61。下载最新的NPM包以获取更新。

别忘了我们是完全开源的！请访问我们的Reaction Native for Windows或Reaction Native for Mac GitHubs，了解更多信息、提交问题、发表意见或提出问题。
