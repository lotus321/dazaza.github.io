MyLinkVault：多终端共享的社会性网络书签站点

2008-09-12 08:37:44

很多人拥有多台PC，办公室的，家里的，临时上网地点的。往往在这些电脑上都收藏了许多的书签。在不同的PC间，书签没有办法同步。唯一的方法，是使书签保存在网上。大家都知道大名鼎鼎的美味书签：del icio。而MyLinkVault更加强调在多台PC间的书签共享。

LOGO

　　MyLinkVault的界面设计成为将许多书签集合于一个单独的网页，很容易修改和重新排序。页面的设计充分采用了web2.0规范。不同的书签分类，可以很方便地用鼠标拖动。收藏一个书签对MyLinkVault来说是最简单的方式。只要把收藏连接拖放到你的浏览器工具栏，将自动生成一个收藏按钮，然后在需要保存的网页中点击一下工具栏上的收藏按钮。就可以把书签加入MyLinkVault。

# MyLinkVault - Go to MyLinkVault

# Save this Link - Save a link, letting you change details before it saves

# QuickLink - Save a link quickly, using the title page as the link name

　　这样，在多台PC间就实现了书签的在线保存和共享。

界面

　　然而社会性的原则是分享。我们来看看MyLinkVault在分享中是怎么做的。
