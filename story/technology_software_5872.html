微软推出web设计软件beta版

by @geekzhang on 2006-09-07 22:45:03

geekzhang 写道 "Microsoft 周三推出了其web设计工具－Expression Web Beta 1(开发代码为“Quartz”)与著名的Adobe Dreamweaver展开竞争。此软件支持CSS的层设计、web规范标准以及对ASP.NET2.0技术的集成，并作为先前微软的web设计工具FrontPage的替代品，XP SP2/Vista的用户现在可以免费安装Expression Web Beta 1，需要 .NET Framework2.0的支持。同时 Expression Graphic Designer / Expression Interactive Designer 也提供了下载。"
