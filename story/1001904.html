GitHub CLI允许您关闭、重新打开问题和公关，以及向问题和公关添加元数据

2020-05-13 09:01:11

GitHub CLI 0.8使得从终端处理拉请求和问题变得更加简单。此版本包括两个主要功能：

您不再需要在创建问题或拉入请求后立即在浏览器中打开问题或拉入请求，而只需添加元数据。现在，您可以在创建拉入请求和问题时添加审阅者、标签、受理人、项目和里程碑(如果适用)。

关闭和重新打开拉入请求和问题，从CLI直接使用gh pr关闭、gh pr重新打开、gh问题关闭和gh问题重新打开。
