苹果推出Logic Pro X以来最大更新Logic

2020-05-12 23:05:45

加利福尼亚州库比蒂诺-苹果公司(Apple)今天发布了Logic Pro X的重大更新，推出了专业版的Live Loops，完全重新设计的采样工作流程，以及新的节拍制作工具。凭借其强大的创意功能集合，Logic Pro X 10.5将是所有音乐家的大规模发布，包括那些制作电子音乐的音乐家。

苹果应用产品营销副总裁苏珊·普雷斯科特(Susan Prescott)说：“Mac和Logic Pro X是世界顶级音乐家和制作人用来创作我们都喜欢的音乐的必备工具。”Logic Pro X 10.5是Logic Pro X推出以来对Logic进行的最大更新，其强大的新工具将激励每一位艺术家--从那些刚刚开始使用Logic的艺术家，到那些已经用它制作格莱美奖获奖专辑的艺术家。我们迫不及待地想知道这些艺术家下一步会创作什么。“。

专业音乐人和制作人已经表达了他们对Logic Pro X 10.5的兴奋之情，他们说：

“Logic Pro X一直是我唯一的Daw。工作流程是无与伦比的，从一开始，内置的音频库对我的音乐来说就是必不可少的。现在，随着快速采样器和鼓机设计师的加入，我恢复了过去在录音室里花在音响和工具包上的时间。这让我可以花更多的时间写新的诗句和编辑70个镜头的人声。“--格莱美奖获奖制作人芬妮亚斯·奥康奈尔：比莉·艾利什、”坏人“和专辑”当我们都睡着了，我们要去哪里？“。

Logic Pro X 10.5改变了我们制作音乐的方式。我们一直在使用所有的新功能，从用MacBook Pro的内置麦克风直接录制声音到快速采样器，到快速编曲，同时使用Live Loops网格录制艺术家。多年来，我们的所有作品都使用了逻辑，新版本将使我们更快地创作持久的歌曲。“-丹泽尔·巴蒂斯特(Denzel Baptiste)和大卫·比拉尔(David BIral)，格莱美提名制作人：Lil Nas X，”Panini“；Sheck Wes，”Mo Bamba“；特拉维斯·斯科特(Travis Scott)和基德·库迪(Kid Cudi)，”The Scotts“；Juice WRLD，”传奇“(Legends)；杜瓦·利帕(Dua Lipa)，”床上好人“；YBN。

有了Mac上的Live Loops，Logic用户现在可以用新的自由形式和非线性方式创作音乐。循环、样本和录音可以组织到一个新的音乐网格中，音乐家可以自发地表演，并将不同的编曲想法捕捉到时间线中。在此基础上，可以使用Logic中的所有专业制作功能进一步细化曲目。

ReMix FX通过一系列激动人心的电子效果(如Bitcrusher、Filter、Gater和中继器)增强了Live Loops，这些电子效果可以在单个曲目或整个混合歌曲中实时执行。当使用免费的Logic Remote应用程序时，这两个功能都变得更加强大，允许用户将他们的iPhone或iPad与他们的Mac配对，以提供对Live Loops和Remix FX的多点触摸控制。

采样器代表了广受欢迎的行业标准EXS24插件的下一代，具有全新的现代设计和扩展的声音整形控制，同时保持完全的向后兼容性。生产商可以使用Ssamer创建和编辑复杂的多采样仪器，使用可自动执行复杂生产任务的优雅拖放工作流程。

快速采样器是一种快速而简单的方法，可以将任何单独的声音转换为可演奏的乐器。音乐家可以从Logic、Finder、Voice Memos中挑选声音，甚至可以直接录制到快速采样器中。只需单击几下，就可以在键盘控制器上修剪、循环和播放导入的样本，并可以访问创造性的声音整形控件。

Logic Pro X 10.5提供了一系列新的创意工具，旨在协同工作，使其快速而流畅地构建原创节拍-嘻哈和电子音乐制作不可或缺的一部分。

Step Sequencer是Logic中的一个新编辑器，旨在使用受经典鼓机工作流程启发的界面，轻松有趣地编程鼓拍、低音线和旋律部分。Step Sequencer将基于模式的音乐创建样式与用于创建原始部分的强大编辑选项相结合-提供对音符速度、重复、选通、跳过、播放方向和随机化的详细控制。

鼓合成提供了广泛的收集踢鼓，圈套，手鼓和打击乐声音完全由软件产生。每种声音都伴随着专门的音形控制，帮助音乐家微调他们最喜欢的部分。鼓合成器声音可以在鼓机设计器的任何垫子上使用，在那里它甚至可以与样本混合，以实现创造性的声音设计选项。

LogicTM中用于构建电子鼓套件的工具Drumer Machine Designer已得到增强，可与新的采样和拍子编程工作流程集成。每个鼓垫可以容纳新的快速采样器和鼓合成器插件，使得在鼓套件中编辑和塑造单独的声音变得容易。套件创建后，可以实时播放，也可以使用Step Sequencer进行编程。

Logic Remote是一款免费的配套应用程序，允许用户将iPhone或iPad与他们的Mac配对，使用多点触摸来控制和执行Logic功能和仪器。Logic Remote正在接收一个重大更新，允许在Live Loops中触发声音、浏览和添加循环，以及将Remix FX应用到会话。

Logic Pro X已经过优化，可以利用最新的Mac硬件和MacOS的强大功能，在处理要求最苛刻的项目时提供令人难以置信的性能，这些项目有数千首曲目和数百个实时插件。

苹果在1984年推出了麦金塔，给个人技术带来了革命性的变化。今天，苹果凭借iPhone、iPad、Mac、Apple Watch和Apple TV引领世界创新。苹果的五个软件平台-iOS、iPadOS、MacOS、WatchOS和TVOS-在所有苹果设备上提供无缝体验，并为人们提供突破性的服务，包括App Store、Apple Music、Apple Pay和iCloud。苹果公司的10万多名员工致力于制造地球上最好的产品，并致力于让世界变得比我们发现的更好。
