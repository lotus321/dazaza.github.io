Windows Live Writer 2009 发布

2009-01-24 17:53:07

昨天微软发布了 Windows Live Essentials，其中的 Windows Live Writer 已经升级为 Windows Live Writer 2009. 新版本的 WLW 对于我来说最大的亮点就是支持搜索草稿和已发布文章，以前都是在 Windows 的搜索里找文章，很麻烦，这个功能终于算是加上去了。
