【下载】WordPress 2.7 RC1 中文版

by @yinyin on 2008-12-02 10:18:22

WordPress 官方刚刚发布了 WordPress 2.7 的最新的测试版，WordPress 2.7 RC1。这是 WordPress 2.7 系列的第4个公开测试版。根据这一版本，我们WordPress 非官方中文站在第一时间制作了 WordPress 2.7 RC1 中文版，提供给 WordPress 中文爱好者进行测试。

WordPress 是目前世界上使用最为广泛的开源的独立博客程序。使用 WordPress，你可以轻松建立自己的博客站点。WordPress 2.7 也是 WordPress 开发历史上变化最大的版本之一，甚至有人疑惑，为什么不叫作 WordPress 3.0？

WordPress 2.7 RC 版也是既 beta3 之后的第四个公开测试版。虽然已经进入了 RC 阶段，但这仍然是测试版。因此，我们建议你不要将此版本直接安装在正式环境中，并在升级此版本之前做好备份。

gz格式：http://wordpresschina.googlecode.com/files/wordpress.27.rc1.chs.tar.gz
