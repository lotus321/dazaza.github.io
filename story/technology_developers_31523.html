20多个XML工具或资源

by @cacard on 2007-10-08 16:30:47

XML 是用户使用的网站众多元素之一，但是很少人懂得它。我们这里收集了20多个工具或者资源，帮助你很好的使用XML（注：以下链接均为英文网站）。

Altova.com － 提供了一整套XML编辑工具的一家商业公司。

CafeConLeche.org － 大量XML相关资源、即时新闻、推荐阅读列表等。

Don’t Invent XML Languages －XML 虽然很容易定制，这个网站告诉你为何最好要使用五种版本中的一种。

eXist － 一个开源的 XML 数据库。

Flash & XML 常见问题 － 有关 Flash 与 XML 之间通讯的常见问题列表以及答案。

JiBX － 用XML创建 Java对象的一个框架。

Open XML Editor －　一个免费XML编辑工具，带有表单测试和验证。

STG XML 验证 － 一个免费的XML文档验证工具。

Tizag XML 向导 － 有关XML各个方面的向导教程。

Vex － 一个可视化的XML编辑器，界面就像字处理工具。

XML W3C 标准 － XML 官方标准。

XML和数据库 － 用XML作为数据库系统的一个参考。

一个XML 检测和验证工具。

XML Cooktop － 一个运行于Windows下的免费XML编辑工具。

XML 拷贝编辑器 － 一个免费的XML复制编辑器，内置验证功能。

XML 开发者中心 － 包括下载，教程，社区等。

XMLFiles.com － 各种教程，例子，链接等。

XMLmind XML Editor － 一个提供免费和收费版本的XML编辑工具。

XML 模式教程 － 来自 W3Schools的教程。

XML.com － XML 各个方面的文章和教程。
