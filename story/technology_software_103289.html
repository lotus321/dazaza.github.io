Safari 更新到 3.2

by @yinyin on 2008-11-14 12:52:38

苹果今日推出了为 Mac 与 Windows PC 设计的世界上最快浏览器:Safari 3.2,网页载入比 Internet Explorer 7 快 1.9 倍、比 FireFox 2 快 1.7 倍,Safari 3.1 执行 JavaScript 的速度是其它浏览器的 6 倍. 供 Mac OS X 与 Windows 用户免费下载,Safari 3.1 也支援最新的网络标准以体验更佳的 Web 2.0 服务.Safari 3.1还是最先支持HTML5视频和音频标签以及CSS动画,CSS Web字体的浏览器,新增防止网络钓鱼功能.
