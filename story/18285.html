Windows Live Gallery升级:成为被浏览最多的个性化网站

2007-05-01 07:13:18

Windows Live Gallery(http://gallery.live.com),由于有三千九百万名不同用户对二千七百万个页面的访问被Nielson统计为世界上被访问最多的个性化站点,这次更新提供了更多新的内容和界面.请看其现在的新特征与新界面!

Windows Live Messenger个人表现 – 现在你可以从Windows Live Gallery包括的表情、图片（aka avatars）、背景等中获取所有个性化的需要。我们会发起一些内容和用户的样品，我们的参与者会增加其内容。

用户体验升级 － 在收到大量用户反馈之后，我们已经改变了社区导航的样式。这将使得用户能够更容易地获取他们所需要的信息。

Windows Live Gallery 参与者层 － 我们将会帮助我们的发表者们在用户体验上与他人区别。我们希望使得我们的参与者能够提供高质量的内容给我们的用户。其好处包括了视觉的微分网络以及上传大量文件的权限。我们将在一下几个月内报道更多关于Windows Live Gallery的好处。
