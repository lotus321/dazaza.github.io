Mozilla 被指抄袭 Web 设计公司的网站设计

by @comsharp on 2010-03-10 14:44:44

设计抄袭是一个经典的话题，毕加索说过，好的艺术家复制，伟大的艺术家剽窃，Web 或 UI 设计这种会随着时间改变的东西，抄袭更是不在话下，国内就不必说了，这种事，微软都不能免俗。最近，Mozilla 被指在一款产品的 UI 设计中全盘照抄一家 Web 设计公司的网站设计，而这家设计公司曾为 Mozilla 提供过设计提案，不过被Mozilla否决了。
