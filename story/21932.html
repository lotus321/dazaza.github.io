今天，微软下载中心出现了一份名为"Windows Automated Installation Kit Documentation (Windows Server code named "Longhorn" & Windows Vista SP1 Beta 3)"的文档，不止一次（Long Zheng说的，但是我只看到了一次……）清晰的提到了Vista SP1，由此可见Vista SP1离我们不远了。

下载到的压缩包中的RTF文档日期是3月，而文档中写的日期是2月，真是混乱，不晓得究竟哪个是对的了。

遍历文档之后发现，没发现有关SP1的新功能信息，不过无人值守安装是有的。
