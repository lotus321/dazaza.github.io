Windows Live Gallery Beta 今日上线

by @Wangtam on 2006-07-24 09:23:11

Windows Live 家族又多了一个新的成员：Windows Live Gallery，如之前我所提过的，这个站点是一个 Apps 资源，有着大量的 Gadgets。Gadgets 原本有一个集中性的站点 Microsoft Gadgets，类似于 Moz 基金会自由的 addons，促进用户自发性的研究、使用和发表作品。当然了，Live Gallery 还不仅仅于此，将要发布的 Live Spaces 中也会大量支持自定义通用组件，比如 Gadgets，包括 Live Messenger，我也在之前的日志里说过新版版可以支持相关扩展，以及其他 Windows Live 服务的个性化插件和扩展都会纳入Live Gallery。...
