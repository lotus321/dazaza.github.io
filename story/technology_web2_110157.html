WordPress 2.7 RC2 中文版发布

by @yinyin on 2008-12-10 09:54:05

WordPress 开发者刚刚发布了 WordPress 2.7 的第二个 RC 测试版: WordPress 2.7 RC2。根据这一版本，我们 WordPress 非官方中文站在第一时间制作了 WordPress 2.7 RC2 中文版，提供给中文社区的 WordPress 爱好者。

WordPress 是一款功能强大的个人信息发布平台。使用 WordPress ，你可以快速地建立自己的个人博客。WordPress 2.7 是 WordPress 一个重要的新版本，这一版本采用全新的用户界面设计，并增加了20个新功能，给用户带来更好的使用体验。

WordPress 2.7 RC2 依然是测试版本，仅供热心网友尝鲜，因此我们提醒大家，不要将此版本直接使用在正式环境中。

WordPress 2.7 RC2 中文版下载：

zip 格式：http://wordpresschina.googlecode.com/files/wordpress.27.rc2.chs.zip

gz 格式：http://wordpresschina.googlecode.com/files/wordpress.27.rc2.chs.tar.gz

WordPress 2.7 RC2 中文语言包下载：

zip 格式：http://wordpresschina.googlecode.com/files/wordpress.27.rc2.chs.languages.only.zip

gz 格式：http://wordpresschina.googlecode.com/files/wordpress.27.rc2.chs.languages.only.tar.gz

WordPress 2.7 演示平台：

为了方便网友体验 WordPress 2.7，我们还建立了一个公共的测试平台，wp27.wpchina.org。
