WordPress 2.7 beta2中文版发布

by @yinyin on 2008-11-07 14:42:38

WordPress 开发团队刚刚发布了 WordPress 2.7 的第二个公开测试版：WordPress 2.7 beta2。根据这一版本，我们 WordPress 非官方中文站制作了最新的 Wordpress 2.7 beta2 中文版，现在提供给网友们下载测试。

WordPress 2.7 beta2 版本主要修正了上一个版本中存在的问题。不过由于这仍然是一个测试版，我们希望网友不要将此版本直接用于正式网站中。WordPress 2.7 beta2 中文版的下载地址为：http://wpchina.org/downloads/ ，你也可从以下地址直接下载：WordPress 2.7 beta2 中文版：http://wordpresschina.googlecode.com/files/wordpress.27.beta2.chs.ziphttp://wordpresschina.googlecode.com/files/wordpress.27.beta2.chs.tar.gzWordPress 2.7 beta2 中文语言包http://wordpresschina.googlecode.com/files/wordpress.27.beta2.chs.languages.only.ziphttp://wordpresschina.googlecode.com/files/wordpress.27.beta2.chs.languages.only.tar.gz此外，我们还建立了一个 Wordpress 2.7 演示站点，供不方便自己建立博客进行测试的网友们进行体验。WordPress 2.7 演示站点的地址为：wp27.wpchina.org。欢迎广大网友进行测试。如果你对 Wordpress 2.7 有特别的感受，也欢迎投稿到 WordPress 非官方中文站。
