Windows 7 SP1 RC版,微软正式提供官方下载

by @reed077 on 2010-10-27 13:43:53

今天微软毫无征兆地发布了Windows 7 SP1和Windows Server 2008 R2 SP1的RC候选版本，近段时间，Windows 7 SP1 RC版本的builds不断被泄露到网络中，可能就是由于这个原因微软才不得已提前对外Windows 7 SP1 RC版。

Windows 7 SP1能帮助PC机利用上述基于服务器的功能，从而为终端用户提供更丰富的VDI体验。此外，SP1并没有新增专门针对Windows 7的功能。所有运行正版Windows 7 RTM操作系统的用户都能够免费安装SP1。
