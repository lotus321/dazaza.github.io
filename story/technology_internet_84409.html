iconfinder -最好的图标搜索引擎

by @hexiaoyao on 2008-08-13 15:24:55

iconfinder是一个搜索引擎的图标站。网站所有者声称，它是最好的图标的搜索引擎。

搜索是基于标签制度。 每个图标有标记。每次搜索将匹配所标记的图标。任何人都可以编辑的标记，使其更相关的图标本身。

对于软件或 Web 界面的设计人员来说，如果能在自己的设计中融入一套漂亮得体的图标，不仅会让 UI 看起来更活泼生动，同时对用户体验的增强也很有益处。网上可免费使用的图标资源非常丰富，但是想找到它们确是很不容易。Iconfinder 就是一个专注于图标的搜索引擎，通过它你可以快速找到你所需要的图标。

Iconfinder 在设计时显然借鉴了 Google 那种极致简洁的风格，打开它的主页，除了一个搜索框外就什么都没有了。但可别小看这个小框框，通过它你就可以快速找到无数漂亮的图标。最令人兴奋的是，这些图标都是基于 CC GPL 或是 LGPL 协议发布的，因此我们可以自由的将他们应用到我们自己的作品中，而不必担心版权问题。

Iconfinder 的搜索框中使用了类似谷歌“搜索建议”的功能，通过 Ajax 技术实时提示相近关键字和匹配结果。Iconfinder 的输出结果支持根据格式过滤，默认将输出从 12*12 到 128*128 所有大小的图标，当然，用户可以通过点击右上的 Select Icon Size 来过滤获得特定大小的图标，很方便。

我尝试从 Iconfinder 下载了几个图标，我发现它所提供的图标大多是透明背景的 PNG 格式图标，这对于我们使用来说是非常方便的。透明 PNG 图片可以很轻松的修改加工成为我们需要的样式，也可以通过 IconFactory Workshop 等等软件方便的转化成透明背景的 ICO 图标，对于设计人员来说在使用时不会有什么障碍。

Iconfinder 实质上也是一种信息纵向搜索，但 Iconfinder 选择的市场切入点很独特。对于流行的搜索引擎来说（例如 Google 和百度）对于像图标这种特定格式具体信息搜索基本上是个盲区，即便我们在图像搜索中使用 filetype:png keywords 这样的格式，也很难搜索到我们需要的图标，Iconfinder 很巧妙的迎合了这部分需求。但 Iconfinder 面向的受众基本上是设计人员，受众较小，因此这样的站点在发展上，我认为应该走精品化路线，如果能与 deviantArt，Iconbuffet 这样的优秀资源合作，使图标质量提高，相信 Iconfinder 将能够成为设计人员收藏夹中不可缺少的一个网站。
