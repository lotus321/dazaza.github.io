可安装在U盘上的操作系统 Puppy Linux 4.00 发布

2008-05-05 19:42:28

在 Puppy Linux 3.01发布六个月后,最新版的 Puppy Linux 4.00 和大家见面了。Puppy Linux 4.00 不到88MB大小，但它自带了大量的描扫仪、打印机、数码相机的驱动以及音乐录制/编辑/转换软件。Puppy Linux 4.00 甚至还包含了一个完整的办公环境。小巧是 Puppy Linux 最大的特色，它小到足以在早期的电脑的内存中运行。Puppy 几乎能在所有的硬件上运行。
