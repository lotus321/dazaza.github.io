树莓派(Respberry Pi)USB无线网卡配置方法

2013-06-30 18:45:25

Raspberry Pi(中文名为“树莓派”,简写为RPi)是为学生计算机编程教育而设计，只有信用卡 大小的卡片式电脑，其系统基于Linux。安装过程请搜索相关文章，如果你使用了 USB Mini 无线网卡，可参考下面的配置方法。

本人的USB无线网卡采用的是：EDUP EP-N8508GS，售价29元。如果你采用DHCP方式动态获取IP地址不成功的话，可参考下面的方法。

使用 lsusb 命令查看有无usb wifi 芯片；

使用 iwlist wlan0 scan 扫描无线网络；

配置 sudo nano /etc/network/interfaces，如下：

auto lo

iface lo inet loopback

iface eth0 inet dhcp

# static ip

auto wlan0

allow-hotup wlan0

iface wlan0 inet static

address 192.168.1.239

getway 192.168.1.1

wpa-ssid "你的SSID"

wpa-psk "你的WIFI密码"

# route

up route add default gw 192.168.1.1

down route del default gw 192.168.1.1

重启即可。
