Google 极力推动 Web 矢量图技术 - 呐喊观察 - 呐喊网络部落格

by @fearless on 2009-10-10 08:10:46

至少在10年前，人们就建议对 Web 图形技术进行改造，而 Google 认为，现在已经是时候了。Google 在昨天10月2日主持召开了 SVG Open 2009 大会，讨论有关 SVG （Scalable Vector Graphics） Web 图形标准的问题。随着众多浏览器厂商以及 Web 开发者对 SVG 的支持，这种可用于 Web 的矢量图形标准很有可能在最近几年成为主流。正如 Google 的开发工程师 Brad Neuberg 在大会中所言，新的编程标准虽说很难形成，但它们确实来了，他说，一开始它们被忽视，接着被炒作，接着被宣布死亡，最后发现，它们是真正能解决问题的技术。矢量图使用直线，曲线，几何图形，色彩值表述图形，而 JPG,GIF 一类的点阵图使用点阵描述图形，前者拥有更小的文件尺寸，不变的缩放质量，比如，我们可以看一下 SVG 版的 Wikipedia 标志，即使放大到整页大小，质量也不下降。
