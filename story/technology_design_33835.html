免费CSS布局和模板大集合!

by @cacard on 2007-11-09 18:19:47

Internet 上有很多基于 (X)HTML/CSS 标记的模板。如果你是一个 Web 开发人员，你不希望把时间一次又一次地浪费在重复代码设计上面，这里提供了一个列表，提供了基于 CSS 的免费模板，几乎所有的资源都配有图片以及完整的布局结构。有些资源已经在 diglog 出现过了，这里正好做一个汇集。

Open Source Templates 是一个免费的 CSS 以及 XHTML 开源模板设计社区。

OSWD 是一个免费下载 Web 设计模板以及分享的社区，现在已经有 2080 个设计者。

OSWT 项目是为那些没有任何设计技能的用户快速创建页面而建立的。

MaxDesign 提供了一组 CSS 页面布局，包括 2 栏以及 3 栏布局。

CSSplay 提供了很多免费的很专业的层次布局。大多数都不需要经过授权而可以任意使用，不过，一些需要经过 email 授权！

Mollio 是一组简单的 html /CSS 模板集合。

Six Shooter Media 提供了15个模板（-____-`` 这也拿得出手？不过，还是十分有共享精神地!）。

CSS Design Templates 提供的模板你可以任意修改和使用，不过，必须要注明提供者的网站链接。

CSS Intensivstation 包含了 15 个布局。所有模板都是基于 XHTML 1.0 以及 CSS2 的哦。

CSSFILL 由 Michael Strand 设计的基于 CSS/XHTML 的布局和模板，免费下载。

CSS Frame Layouts 模板，基于CSS的页面模板，用于快速构建你的页面架构。

CSS Template Gallery 提供搜索功能，可惜只有 32 个模板（-_____-```）。

Ex-DesignZ 提供了 300 多个基于 (X)HTML/CSS 的模板。

Jimmy Lin 提供了完全（100%地完全!）基于 CSS/XHTML 以及适用于 IE5.0+ 、FireFox 1.5、Opera8.5 及 Safari 2。没有使用任何技巧（Hacks），并且可以轻而易举地进行修改。没有版权限制，但是必须注明来源链接。

Layout Gala 提供了 40 多个专业布局。使用规范的标记以及 CSS，在当前主流浏览器都能通过测试。

Little Boxes 包含 16 个布局，经典！

Solucija  一个完全免费的基于 XHTML / CSS 的站点模板下载站点。可用户商业化或者非商业化网站，不过，要保留一个网站链接。

The Layout Reservoir 提供 2 栏以及 3 栏的布局。

Linknosie 是一个工具，你可以自定义布局类型，宽度以及其它的参数来生成布局。

Code Sucks 提供了大约 100 个 CSS 布局供下载。所有布局是按照 1024x768 的尺寸设计的。

SSI Develper 提供基于 XHTML 1.0 以及 CSS2 的 CSS 模板。
