Desktop Earth：送你一张动态的“地球壁纸”
厌倦了静态壁纸的呆板？想不想弄一张动态壁纸玩玩？担心占用太多资源？那 Desktop Earth 应该是你尝试的首选。这个软件是将我们的地球搬到桌面，实时显示昼夜交替的变化，而且它是免费的，内存占用6M左右（并不是很夸张）。
AnyClient：随时随地上传下载
AnyClient 提供的是一种在线服务：它模拟出 FTP 软件的界面，这样只需用浏览器打开这个网站，输入 FTP 服务器的用户名和密码即可轻松地进行上传和下载了。当然网站也提供了客户端供下载安装使用，不过这不是它的优势。
