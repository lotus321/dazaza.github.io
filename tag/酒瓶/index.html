1800 Tequila的炫彩艺术酒瓶设计
1800 Tequila Limited是家卖酒的公司。公司不仅关注酒的生产，在酒瓶上也做足了功夫。公司请了很多艺术家帮忙设计酒瓶，还允许用户定制个性酒瓶。如果这公司在中国，我也会去定做个酒瓶玩玩。欣赏一下艺术家们设计酒瓶吧：
“无极全程”酒瓶设计
无极全程是我国的一家酒类包装策划设计的公司，这些酒瓶的设计，就是来自于他们。喜欢上面这个袋子。 Via Designyearbook
Sangria 创意酒瓶音箱
Carlo Rossi 使用酒瓶为新的 7.1 声道音箱做了一个概念设计，内充一些特殊气体，并且保持了良好的音效和保护性，很特别的感觉。目前当然还不可能有投入生产的可能性消息，希望有厂商可以完善这一种设计，并付之投产。[ via SCI FI Tech ]... 继续.
